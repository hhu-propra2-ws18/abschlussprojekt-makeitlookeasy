`File -> Settings -> Editor -> Code Style -> Java -> Import Scheme -> IntelliJ Idea code style XML` +

.Force code formatting on every commit:
`File -> Settings -> Version Control -> And then check the following:`

  * Rearrange code
  * Optimize imports
  * Cleanup
